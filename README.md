# svelte-date-picker

Date and time picker for Svelte

## Dev instructions

### Get started

1. Install Node.js (v14 works)
2. Run `npm install`

### Commands

- `npm run dev`: Start in dev mode
- `npm run build`: Build
- `npm run preview`: Preview production app
- `npm run check`: Run `svelte-check`
- `npm run lint`: Lint
